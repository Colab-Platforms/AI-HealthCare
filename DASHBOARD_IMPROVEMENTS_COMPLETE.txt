## Technical Implementation:

**Mobile (< lg breakpoint):**
```
- flex overflow-x-auto
- gap-3 pb-2
- scrollbar-hide
- snap-x snap-mandatory
- Each card: min-w-[160px] snap-start
```

**Desktop (>= lg breakpoint):**
```
- lg:grid lg:grid-cols-4
- lg:gap-4
- lg:overflow-visible
- lg:snap-none
- Each card: lg:min-w-0
```
